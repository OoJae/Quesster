## ğŸ“± How to Run Locally

### Prerequisites
* Node.js v18+
* Yarn
* A Supabase project

### 1. Clone & Install
```bash
git clone https://github.com/OoJae/Quesster.git
cd mobile-app/packages/react-app
yarn install
```

### 2. Environment Variables
Create a .env.local file in packages/react-app:
```bash
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

### 3. Run Development Server
```bash
yarn dev
```

### 4. Test in MiniPay (using Ngrok)
Since MiniPay requires a secure HTTPS connection, run Ngrok in a separate terminal:
```bash
ngrok http 3000
```

Paste the resulting URL into the MiniPay Developer Settings â†’ "Load Test Page".
